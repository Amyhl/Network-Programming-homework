# 项目6自由设计
## 介绍
基于《Qt及Qt Quick开发实战精解》的局域网聊天软件设计为基础开发的一款即时IM软件。

## 计划安排
首先实现基本聊天功能  
之后仿照同级目录下的Chat与QQ做一个Demo出来  
后续可能涉及:
- [ ] 注册登录功能
- [ ] 导入导出数据库
- [ ] 多线程
- [ ] 界面优化
- [ ] 功能优化
